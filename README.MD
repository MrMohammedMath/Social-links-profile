### **Usage**

```bash
  git clone https://github.com/MrMohammedMath/Social-links-profile.git

  cd Social-links-profile

  npm install
```

- for running dev server

```bash
npm run dev
```

- for build & review

```bash
npm run build
npm run preview
```
